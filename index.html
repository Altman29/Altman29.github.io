原生开发工具包 (NDK) 是一组可让您在 Android 应用中利用 C 和 C++ 代码的工具。 可用以从您自己的源代码构建，或者利用现有的预构建库。
NDK 不适用于大多数初学的 Android 编程者，对许多类型的 Android 应用没什么价值。 因为它不可避免地会增加开发过程的复杂性，所以通常不值得使用。 但如果您需要执行以下操作，它可能很有用：
从设备获取卓越性能以用于计算密集型应用，例如游戏或物理模拟。
重复使用您自己或其他开发者的 C 或 C++ 库。

其实Android 6.0权限适配已经了解很久了，刚出的时候也埋怨过，不过想想也是，之前的权限方案太草率了。不知道大家有没有遇到过莫名其妙扣费的情况，我就遇到过，好久之前用Android上一款单机游戏App需要联网，需要联系人，需要发短信的权限，然后玩了几关后，疯狂的收到扣费短信，玩了一会八十大洋就没了。。也是印象深刻。Android 6.0推出的动态权限管理就是为了解决这一问题，提升用户体验，当然也为了程序员增加了些负担。
